# MktAi
Plataforma que conecta ao Meta Ads e Google Ads para gerenciar campanhas publicitárias de forma autônoma com IA. Inclui criação de campanhas com criativos gerados por IA, testes A/B, otimização de orçamento via fila de jobs (BullMQ + Redis) e integração OAuth com as plataformas de anúncios.  Stack: React · Node.js · Express · PostgreSQL · BullMQ
